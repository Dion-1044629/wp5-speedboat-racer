# WP5 Speedboat Racer

## Doel
Top-down racing game (“Speedboat Racer”) met timer, positie-indicator en 4 AI-tegenstanders.

## Controls
- Gas: W / ↑
- Rem: S / ↓
- Sturen: A / D of ← / →
- (Optioneel) Handrem: Space
- (Optioneel) Reset: R
